# Luxury Portfolio Website

A modern, dark-themed luxury portfolio website inspired by high-end fashion and design portfolios.

## Features

- **Dark Luxury Theme**: Elegant dark color scheme with gold accents
- **Responsive Design**: Fully responsive across all devices
- **Smooth Animations**: Subtle animations and transitions throughout
- **Multiple Pages**:
  - Landing Page with hero section and featured work
  - About Page with biography, skills, and timeline
  - Portfolio Page with filterable projects and resume section
  - Gallery Page with visual showcase
  - Contact Section with form

## Pages

1. **index.html** - Landing page with hero section, featured work, and contact form
2. **about.html** - About page with personal information, skills, and journey timeline
3. **portfolio.html** - Portfolio page with filterable projects and detailed resume section
4. **gallery.html** - Visual gallery showcasing creative works

## File Structure

```
├── index.html          # Landing page
```

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --bg-primary: #0a0a0a;
    --bg-secondary: #111111;
    --accent: #d4af37;
    /* ... */
}
```

### Content
- Replace placeholder text in HTML files
- Add your own images (replace the placeholder divs)
- Update contact information in the contact section
- Customize the resume section with your own experience

### Images
Currently using placeholder divs. To add images:
1. Create an `images` folder
2. Replace `<div class="image-placeholder">` with `<img src="images/your-image.jpg" alt="Description">`
3. Update CSS as needed
